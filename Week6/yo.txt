<h2>Using asyncio</h2> 
<p pid="75">The asyncio module is focused on handling system events. It works around an event loop that waits for an event to occur and then reacts to that event. The reaction can be calling another function. This process is called even handling. The asyncio module uses coroutines for event handling.</p> 
<p pid="76">To use the asyncio event handling and coroutine functionality, we will import the asyncio module:</p> 
<pre><code lang="text/x-python">import asyncio</code></pre> 
<p pid="176"><br /></p> 
<p pid="77">Now, define the asyncio coroutine method like this:</p> 
<pre><code lang="text/x-python">async def coroutine():

    await my_func()</code></pre> 
<p pid="178"><br /></p> 
<p pid="78">The keyword async tells that this is a native asyncio coroutine. Inside the body of the coroutine, we have the await keyword, which returns a certain value. The return keyword can also be used.</p> 
<p pid="79">Now, let's create a code using a coroutine to download a file from the web:</p> 
<pre><code lang="text/x-python">&gt;&gt;&gt; import os

&gt;&gt;&gt; import urllib.request

&gt;&gt;&gt; async def coroutine(url):

    r = urllib.request.urlopen(url)

    filename = &quot;couroutine_downloads.txt&quot;

    with open(filename, 'wb') as f:

        for ch in r:

            f.write(ch)

    print_msg = 'Successfully Downloaded'

    return print_msg

&gt;&gt;&gt; async def main_func(urls_to_download):

    co = [coroutine(url) for url in urls_to_download]

    downloaded, downloading = await asyncio.wait(co)

    for i in downloaded:

        print(i.result())

urls_to_download = [&quot;https://www.python.org/events/python-events/801/&quot;,

&quot;https://www.python.org/events/python-events/790/&quot;, 

&quot;https://www.python.org/events/python-user-group/816/&quot;,

 &quot;https://www.python.org/events/python-events/757/&quot;]

&gt;&gt;&gt; eventLoop = asyncio.get_event_loop()

&gt;&gt;&gt; eventLoop.run_until_complete(main_func(urls_to_download))</code></pre> 
<p pid="180"><br /></p> 
<p pid="80">In this code, we created an async coroutine function that downloads our files and returns a message.</p> 
<p pid="81">Then, we have another async coroutine calls the&nbsp;<code>main_func</code><span id="_tmp_pre_12">&nbsp;</span>that waits for the URLs and makes a queue of all URLs. The wait function of asyncio waits for the coroutines to complete.</p> 
<p pid="82">Now to start the coroutine, we have to put the coroutine inside the event loop by using the&nbsp;<code>get_event_loop()</code>&nbsp;method of asyncio, and finally, the event loop is executed using the&nbsp;<code>run_until_complete()</code>&nbsp;method of asyncio.</p> 
<p pid="83">Downloading files using Python is fun. I hope you found this tutorial useful!</p>
